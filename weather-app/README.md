# Weather App

## Project Description
This is a web-based Weather Application that allows users to enter a city name and retrieve
current weather details such as temperature, weather description, and an icon representing
the weather condition.

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## Features
- City-based weather search
- Displays temperature in Celsius
- Shows weather description and icon
- Error handling for invalid city names
- Responsive and user-friendly UI

## How to Run the App
1. Download or clone the project.
2. Open `script.js` and replace `YOUR_API_KEY_HERE` with your OpenWeatherMap API key.
3. Open `index.html` in any modern web browser.
4. Enter a city name and click **Submit**.

## API Used
OpenWeatherMap API  
https://openweathermap.org/

## Notes
- Ensure you have an active internet connection.
- This project uses the free tier of the OpenWeatherMap API.
